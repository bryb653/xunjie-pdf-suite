## 迅捷 PDF 编辑器怎么用：核心操作教程

### 编辑 PDF 文字

1. 打开软件，点击左上角【文件 - 打开】，导入需要编辑的 PDF 文件
2. 点击顶部工具栏中的【内容编辑】功能键
3. 文件页面上会出现带文字的编辑框，鼠标点击选中目标文字区域
4. 双击进入编辑状态，直接修改文字内容
5. 右键点击编辑框，选择【属性】，可调整字体、大小、颜色
6. 修改完成后点击保存图标，输出修改好的 PDF 文件

### 给 PDF 添加水印

1. 打开文件后，在顶部菜单找到【文档 - 水印 - 添加】
2. 弹窗中选择水印来源（文字或图片），输入水印内容
3. 设置旋转角度、透明度、位置，选择应用页面范围
4. 点击确定，水印添加完成后保存即可

### 删除页面 / 合并 PDF

点击顶部【文档】菜单，找到【删除页面】或【合并文件】选项，操作提示很直接，选好范围点确认就行。学习成本比想象中低。
